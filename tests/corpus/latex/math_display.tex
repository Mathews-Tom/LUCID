\section{Display Math}

The Euler identity is given by the following equation.

\begin{equation}
e^{i\pi} + 1 = 0
\end{equation}

We can also write display math using double dollar signs.

$$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$

This concludes the display math examples.
